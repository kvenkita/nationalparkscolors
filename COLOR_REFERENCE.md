## Usage Tips

### Accessing Colors by Index
```r
# Get first color from Yellowstone palette
natparks_palettes$Yellowstone[1]  # Returns "#F4E04D"

# Get multiple colors
natparks_palettes$GrandCanyon[c(1, 3, 5)]
```

### Creating Custom Palettes
```r
# Combine colors from different palettes
custom_pal <- c(
  natparks_palettes$Yellowstone[1],
  natparks_palettes$Glacier[2],
  natparks_palettes$CraterLake[3]
)
```

### Color Testing
```r
# Display palette
natparks_palette("Yosemite")

# Test in a simple plot
plot(1:5, col = natparks_palettes$Acadia, pch = 19, cex = 3)
```
